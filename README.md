  * カテゴリーページでのログインとニックネーム入力

  * ホームページからスタートボタンで遷移
* ツリー関連機能
  * ツリーのリスト表示と選択

  * アイドルプロフィールの表示
* その他
  * 各種ページのルーティング設定
